AeroMed iOS Application

Overall Goal

We have been given the responsibility of creating an iOS 7 based application to be used by the AeroMed team at Spectrum Health. This application will be used to keep track of procedures and checklists of important vital tasks based on each procedure based on the patients conditions.

1 - Software

Our application is designed for the latest version of Apple's iOS 7 and will be developed within Apple's own Xcode IDE.

2 - Devices

The original project description state that the application would be for iPad's only, upon meeting with them further we will be making it for both iPad and iPhones running iOS 7 only.

3 - User Interface

The UI will be information driven to be able to look up a condition with a provided checklist of what is needed for the procedure.

Prototypes.

Currently there is no name for the application as the application may be used commercially not just internally.
